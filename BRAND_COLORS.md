# Brand Colors

## Primary Identity

**Sovereign Violet** `#5B3DFF` is the signature product color. It represents intelligence, control, authority, and the primary operating layer for enterprise AI.

Use it for:
- Primary CTAs.
- Selected navigation.
- Active tabs.
- Focus states.
- Key recommendation actions.
- Important optimization opportunities.

Use it with restraint. Sovereign Violet should signal a decision or a selected control, not decorate every surface.

## Usage Rules

- Use Sovereign Violet for decisions and selected state.
- Use Infra Cyan only for live infrastructure, agents, online systems, telemetry, and active runtime signals.
- Use Emerald, Amber, and Rose only for health/risk states.
- Use Obsidian/Graphite for command surfaces, not for every page section.
- Use Porcelain/Mist to keep the workspace calm and premium.
- Keep charts restrained: lead with Sovereign Violet, then Infra Cyan and status colors.

## When Not To Use Sovereign Violet

- Do not use it for every icon or KPI.
- Do not use it for warning, critical, or healthy states.
- Do not use it as a large decorative gradient.
- Do not pair it with bright pink/purple gradients that make the UI feel consumer or childish.
- Do not use it where Infra Cyan is more semantically correct, such as live agent or infrastructure telemetry.
