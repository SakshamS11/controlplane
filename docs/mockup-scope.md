# Mockup Scope

AI Control Plane is presented here as a mockup-only SaaS MVP for client, investor, and technical partner demos.

The app demonstrates:

- enterprise landing page
- command-center dashboard
- deployment target inventory
- Acme target detail demo flow
- unified model catalog
- AI stack templates
- department model access governance
- monitoring dashboards
- audit logs
- settings and connected providers

The app intentionally does not include:

- real remote agents
- backend APIs
- databases
- cloud provisioning
- Kubernetes orchestration
- LiteLLM or vLLM production setup
- SOC 2 workflows
- billing
- SSO
- secrets

All data is mocked in `apps/dashboard/lib/mock-data.ts`.
